# Student Attendance Tracker

## Overview
A student attendance management system created using Google Sheets and Microsoft Excel.

## Features
- Student Name and Roll Number Tracking
- Present and Absent Count
- Automatic Attendance Percentage Calculation
- Clean and User-Friendly Layout

## Skills Used
- Microsoft Excel
- Google Sheets
- Data Entry
- Spreadsheet Management
- Basic Formulas
